# Rippl — Clean Water Initiative (SDG 6)

![Rippl Banner](images/web_img_5.png)

## 🌊 Overview
**Rippl** is a premium digital research platform dedicated to advancing **Sustainable Development Goal 6: Clean Water and Sanitation**. Developed as a group coursework project for the Web Development module at the **University of Westminster**, this website serves as a comprehensive hub for hydrological data, conservation strategies, and community sanitation research.

Our mission is to bridge the gap between complex environmental data and public awareness, proving that **every drop counts and small actions create a big impact**.

---

## 🚀 Key Features

### 📡 Aquatic Integrity System (AIS)
A high-fidelity simulator that tracks global water purity levels. Utilizing satellite telemetry and sensory nodes, the AIS provides real-time visualizations of hydrological metrics across 140+ countries.

### 📚 Research Core
Individualized research modules exploring critical aspects of SDG 6:
- **Water Conservation**: Strategies for industrial and domestic reduction.
- **Quality Monitoring**: In-depth analysis of global purity standards.
- **Community Sanitation**: Focus on rural infrastructure and hygiene programs.
- **Global Accessibility**: Addressing the challenges of water scarcity in developing regions.

### 🎨 The Fluid Curator (Design System)
A modern, responsive design system built with a focus on smooth transitions and a calm, oceanic aesthetic.
- **Typography**: Manrope for bold headlines and Inter for readable body text.
- **Color Palette**: Deep oceanic blues (`#001e40`), vibrant aqua accents (`#2dbcfe`), and clean surface variants.
- **Micro-interactions**: Hover effects and custom-built navigation for an immersive user experience.
